    • Naviadsgasi, Eksekusi, Read, Copy, dan Move - Linux CLI Dasar

    by Pratama on Linux

    Pendahuluan

    Pada bagian ini kita akan mempelajari dasar-dasar penggunaan command line interface (CLI) linux. CLI pada linux sering disebut dengan istilah lain diantaranya: terminal, console,shell, dll. Bagi pengguna komputer pemula yang lebih sering berinteraksi dengan tampilan grafis (GUI-graphical user interface) mungkin hanya bisa terdiam menatap monitor tanpa tahu harus berbuat apa ketika dihadapkan dengan tampilan seperti di bawah ini.

    Penggunaan CLI memiliki beberapa keuntungan jika dibandingkan dengan GUI diantaranya:

    1. Cepat dan efisien ; eksekusi dan navigasi sistem lebih cepat, tombol TAB memiliki fungsi autocompletion.
    2. Minim sumber daya ; selain program itu sendiri, GUI memerlukan lebih banyak sumber daya misalnya me-load elemen icons, fonts, images, driver, dll.
    3. Customisasi ; kemampuan alias, scripting, pipeline, dapat dicustomisasi sesuai kebutuhan.

    Navigasi & Eksekusi pada CLI

    Hal pertama yang harus dipahami dalam penggunaan CLI adalah navigasi, yaitu berpindah dari satu directory(folder) ke directory lainnya, hal ini sangat penting untuk proses selanjutnya yaitu mengeksekusi script/program yang letaknya di dalam folder. Sebagai contoh, pada sistem operasi Windows ketika kita klik shortcut Notepad yang berada di Start > All Programs > Accesories secara tidak langsung kita bernavigasi ke folder C:\Windows\System32\ dan mengeksekusi program notepad.exe.

    Tutorial Navigasi & Eksekusi CLI

    Mari berimajinasi bahwa kita ada di sebuah benteng, besar dan kokoh, sulit ditembus oleh musuh. Kita berjalan menyusuri menara-menara benteng yang menjulang tinggi ke langit, menyusuri taman benteng yang ditumbuhi bunga berwarna-warni. Sampai akhirnya kita menjelajahi lorong bawah tanah benteng yang pengap, lembap, dan gelap.

    • Buka terminal emulator / console dan ketik perintah berikut:
    $ git clone https://github.com/mpratama/clitut.git

    Ketika membuka terminal emulator secara default kita berada di directory /home/{username}/ ditandai dengan tanda ~. Tanda dollar $ tidak perlu diketik, tanda dollar menandakan kita mengakses terminal dengan hak akses user biasa.

    • Ketik cd clitut untuk masuk ke directory clitut, lalu ketik ls untuk menampilkan isi directory dimana kita sedang berada. Pada terminal akan tampil output sebagai berikut:
    $ ls
lorong_bawah_tanah README.md

    Tips: gunakan tombol keyboard TAB untuk menggunakan fitur autocompletion yang sangat membantu. cd berarti change directory sedangkan ls berarti list directory contents.

    • Mari melangkah lebih dalam, masuk ke lorong_bawah_tanah. cd lorong_bawah_tanah
    • Untuk mengetahui isi lorong_bawah_tanah selain dengan cd bisa dengan tree untuk mengetahui isi yang lebih detail.
    $ tree

    Perintah tree berfungsi memetakan seluruh isi directory termasuk seluruh isi subdirectory didalamnya. Perlu perhatian pada penggunaan perintah tree terutama pada directory yang berisi banyak subdirectory dan file.

    • Masuk ke ruang_penjaga cd ruang_penjaga. Berinteraksi dengan lampu menggunakan python lampu.py. Untuk membuka gerbang gunakan sh buka_gerbang.sh
    $ python lampu.py
Nyalakan lampu(y/n): y
Lampu menyala.
Ruangan menjadi terang.
$ sh buka_gerbang.sh
Gerbang terbuka.

    File lampu.py, buka_gerbang.sh, dan tutup_gerbang.sh dalam directory ruang_penjaga merupakan script atau program. Script berakhiran py merupakan python script dan bisa dieksekusi menggunakan perintah python sedangkan script berakhiran sh merupakan shell script dan dieksekusi menggunakan perintah sh. Berbeda jenis script berbeda juga cara mengeksekusinya, misal: .js = javacript, .rb = ruby, .jar = java archive, .lua = lua, .pl = perl, .exe = executable, dll.

    • Kembali ke lorong_bawah_tanah dengan cd ... Gunakan pwd untuk mengetahui posisi sekarang. Masuk ke ruang sel_tahanan dan masuk ke sel_1 cd sel_tahanan/sel_1
    $ cd ..
$ pwd
/home/username/clitut/lorong_bawah_tanah
$ cd sel_tahanan/sel_1
$ ls
tahanan_1

    Tanda titik 2 .. berarti naik satu tingkat ke directory sebelumnya. pwd berfungsi untuk print name of current/working directory.

    Read, Copy dan Move File

    • Dari ruang sel_1 loncat ke gudang_makanan cd ../../gudang/gudang_makanan/. Untuk mengetahui persediaan makanan baca file teks dengan cat persediaan_makanan.txt. Untuk menyalin catatan persediaan makanan dan menyimpannya di ruang penjaga gunakan cp persediaan_makanan.txt ../../ruang_penjaga/
    $ cd ../../gudang/gudang_makanan/
$ cat persediaan_makanan.txt
beras   : 200 kilo
daging  : 450 kilo
cabai   : 300 kilo
susu    : 500 liter
gandum  : 250 kilo
$ cp persediaan_makanan.txt ../../ruang_penjaga/

    Perintah cat berfungsi membaca isi textfile/script dan menampilkannya pada output. cp berfungsi membuat salinan satu atau beberapa file, format penggunaannya yaitu cp file1 file2 file3 filedst target_penyimpanan/

    • Dari gudang_makanan pindah ke gudang_senjata cd ../gudang_senjata/. Berinteraksi dengan anjing_penjaga dengan perl anjing_penjaga.pl.Baca catatan persediaan_senjata cat persediaan_senjata.txt lalu salin ke ruang_penjaga cp persediaan_senjata.txt ../../ruang_penjaga/. Pindahkan anjing_penjaga ke sel_tahanan mv anjing_penjaga.pl ../../sel_tahanan/.
    $ cd ../gudang_senjata/
$ perl anjing_penjaga.pl
Woof.. woof..
$ cat persediaan_senjata.txt
pedang  : 100
tombak  : 100
busur   : 100
panah   : 100000
peledak : 75
$ cp persediaan_senjata.txt ../../ruang/penjaga/
$ mv anjing_penjaga.pl ../../sel_tahanan/

    File berakhiran pl merupakan perl script. Sebelumnya sudah dijelaskan bahwa berbeda script maka berbeda juga cara mengeksekusinya. Untuk perl script dapat dijalankan dengan perintah perl. Perintah mv memiliki fungsi untuk moving files, selain itu juga berfungsi untuk mengubah nama file misal mv anjing_penjaga.pl nymeria.pl

    Penutup

    Di atas adalah contoh-contoh penggunaan CLI pada sistem Linux. Contoh yang dijelaskan di atas hanya berupa dasar-dasar, masih banyak perintah, teknik, dan juga customisasi lainnya. Praktik sendiri, eksplore, cari literatur yang banyak. Karena dengan menguasai CLI akan meningkatkan efektifitas dan menambah skill programming, selain itu juga terlihat keren ketika orang lain melihat kita mengutak-atik komputer melalui CLI. Oh iya.. satu tips yang cukup penting.. jangan lupa gunakan tombol keyboard TAB untuk membantu.
