    Pengantar

    Dalam kehidupan sehari-hari kita tidak pernah lepas dari yang namanya komputer, laptop, smartphone, dan internet. Zaman semakin berkembang dan kita telah sampai pada satu titik dimana setiap harinya kita membutuhkan bantuan dari teknologi. Untuk dapat memanfaatkan teknologi secara maksimal maka kita harus mengetahui bagaimana cara kerja teknologi tersebut, dan salah satu caranya adalah dengan pemrograman komputer.

    Pada website ini akan diposting banyak tulisan yang akan mengajak kamu mempelajari programming dengan berpetualang ke negeri dongeng. Mengarungi lautan, menggunakan sihir, megalahkan monster, sampai menyelamatkan sang putri. Semuanya menggunakan python programming. Siapkah kalian untuk memulai petualangan ini. Mari kita mulai!!

    Naikkan jangkar, bentangkan layar - Install python

    Baiklah—inilah saatnya untuk membuka peta dunia dan berlayar menuju negeri Pythonista—disana kita akan meminta kepada penduduk negeri itu sebuah benda yang konon bisa mengeluarkan sihir jika kita menggunakan bahasa tertentu—banyak para kesatria, assasins, hingga wizard dari seluruh negeri mengunjungi negeri Pythonista untuk mendapatkan benda ini untuk mengatasi segala masalah yang dihadapi.

    Python gratis untuk siapa saja, tetapi untuk mendapatkannya memerlukan koneksi internet untuk mendownloadnya. Gunakan web browser dan segera berlayar ke Negeri Python. Ada beberapa cara berbeda untuk install python, tergantung mesin yang kamu gunakan—tapi dalam tulisan ini asumsi kamu menggunakan mesin Windows. Pastikan versi python yang kamu download adalah versi 3.x.x. Proses intalasi cukup dengan double-click python installer dan ikuti petunjuk sampai selesai.

    Setelah proses instalasi selesai pastikan python sudah terinstall di sistem dengan cara klik Start menu dan klik cmd.exe lalu masukkan perintah dibawah ini:

    C:\windows\system32>python -V

    Jika python sudah terinstall akan keluar *output* Python berikut versi sesuai yang terinstall

    Python 3.4.0

    Misi Berhasil : Menginstall Python!

    Sempurna! Kita telah mendapatkannya! Benda ajaib yang dimiliki penduduk Pythonista tersebut adalah sebuah tablet legendaris berkemampuan sihir jika kita menggunakan bahasa tertentu. Kita bisa mulai mempelajarinya. Pada BAB berikutnya kita akan mempelajari basic python dan menggunakan python secara interaktif, dimana python akan menjalankan kode yang kamu tulis hanya dengan menekan Enter.

    Basic: menjalankan python interaktif

    Untuk menjalankan python secara interaktif caranya dengan membuka cmd.exe dan memasukkan perintah python sebagai berikut:

    C:\windows\system32>python

    Maka kamu akan melihat tampilan python interaktif seperti ini:

    Python 3.4.0 (default, Apr 11 2014, 13:05:18)
[MSC v.1500 32 bit (Intel)] on win32
Type "copyright", "credits", or "license()" for more information.
>>> print("Halo dunia!")
Halo dunia!
>>>

    Tanda panah tiga >>> adalah dimana kamu menuliskan perintah python. Seperti contoh di atas perintah print("Halo dunia!") akan mengeluarkan output sesuai kata yang kamu tulis di dalam tanda kurung.

    Basic : Menggunakan text editor

    Cara lain menjalankan perintah python adalah dengan menuliskan script, yaitu menuliskan baris demi baris perintah pada sebuah script lalu menjalankannya sekaligus bukan satu-per-satu. Untuk menuliskan script kamu membutuhkan aplikasi text editor (Jangan keliru dengan Microsoft Word—yang merupakan aplikasi word processor—aplikasi untuk menulis cerita atau laporan sekolah, tapi bukan aplikasi yang bagus untuk menulis program komputer.)

    Program notepad merupakan salah satu aplikasi text editor, namun program notepad bawaan windows kurang bagus untuk menulis program komputer. Alternatif aplikasi lain salah satunya adalah [notepad++](https://notepad-plus-plus.org/).

    Setelah menginstall apikasi notepad++, buka dan ketikkan print("Halo dunia!") seperti pada saat mencoba python interaktif. Lalu simpan dimanapun (misal di Desktop) beri nama halo.py (tidak masalah mau diberi nama apapun asalkan diakhiri dengan format .py—hindari spasi dalam penamaan).

    Untuk menjalankannya dengan membuka cmd.exe lalu ganti lokasi tempat kita menyimpan script—contoh di atas di Desktop. Cara mengganti lokasi pada cmd windows adalah menggunakan perintah cd:

    > cd C:\Users\username\Desktop

    Jangan ketik tanda >, ketikan perintah selanjutnya (dan juga jangan ketik \username\ seperti contoh di atas, ketik sesuai user pada komputer kamu, misalnya di komputer saya > cd C:\Users\Administrator\Desktop).

    Setelah berganti lokasi di Desktop, untuk menjalankan script ketikkan:

    > python halo.py
Halo dunia!

    Hal-hal di atas merupakan basic python, mulai dari: instalasi, python interaktif, sampai dengan menjalankan kode python dari script. Pada tulisan berikutnya akan dijelaskan cara menggunakan python lebih lanjut.
